### Basic Controls  
| Key                | Action                            |
| ------------------ | --------------------------------- |
| `W`, `A`, `S`, `D` | Move the player                   |
| `Space`            | Jump                              |
| `F`                | Toggle flying on/off              |
| `O`                | Respawn the world with a new seed |
